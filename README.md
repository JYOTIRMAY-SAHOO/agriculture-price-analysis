# 🌾 Agriculture Price Data Analysis & Prediction

## 📌 Overview

This project analyzes agricultural commodity prices using Python and predicts prices using Machine Learning techniques. It helps in understanding price trends across different states and commodities.

## 🎯 Objective

* Analyze agricultural price data
* Identify trends and patterns
* Build a predictive model for price forecasting

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Features

* Data Cleaning & Preprocessing
* Data Visualization (Bar Chart, Heatmap, Boxplot)
* Time Trend Analysis
* Price Prediction using Random Forest

## 🤖 Machine Learning Model

* Model: Random Forest Regressor
* Input Features:

  * Commodity
  * State
  * Min Price
  * Max Price
* Target:

  * Modal Price

## 📈 Results

* Strong correlation between price variables
* Certain states show higher prices
* Model achieves good prediction accuracy

## 📁 Dataset

The dataset includes:

* Commodity
* State
* Arrival Date
* Min Price
* Max Price
* Modal Price

* Real-time price prediction

* Weather-based analysis
